COMP 1950 Project: Meta Part

Home Page

The Home page includes the main functions and information. Only the enrolled students in the course have the permission to access the course web page. At a quick glance at the dashboard, the student can view his/her homework, quiz grades, upcoming course/lecture material, course information and any instructor notifications. In general the page is divided into three main columns.

  1. The left column has a side navigation bar to access the course information, lectures, exercises, homework, quizzes, archived notes (notes taken by the student in the class or when reviewing the lecture), webspace and resources.
  2. The middle column displays the homework, quiz and assignment grades, upcoming course/lecture material, course information and any instructor notifications.
  3. The right column is divided into two parts: Notes and Chat area. Notes is activated when the student is logged in and viewing a lecture.

Display Lectures

In this page the student can access the activated lectures by the instructor. For user experience, the lecture contents will be displayed in an accordion style to save space and eliminate the scrolling to get to the bottom of the lecture.

Also, the student can expand or collapse the lecture if he/she would like to access the sections. The download facility is provided for the student for each lecture.

Utilizing the HTML5 editable fields feature, students can take notes while on a lecture or educational page. The notes for each lecture can be saved and an archive of all the notes can be found in the "Archived Notes" page. All the notes are archived for the students, so they can download or retrieve them at any time. All notes are archived and organized by lecture title.

Homework

In this page the student can access the activated Homework assignment. Homework contents will be displayed in an accordion style to save space and eliminate the scrolling. With each homework due date is displayed. Students can download the homeworks from the lecture page or can select the Homework option from the menu to display or download the homeworks.

Exercises

In this page the student can access the activated Exercises only.Exercises can be accessed within the lecture or using this page. Each lecture includes different exercises. The student selects the lecture from the Exercises menu then all exercises of that lecture will be displayed in an accordion style. Students can download the exercise in pdf or word format.
