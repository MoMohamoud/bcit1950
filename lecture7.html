What is JavaScript?

  • From Wikipedia:

    JavaScript is a scripting language most often used for client-side web development. JavaScript was influenced by many languages and was designed to have a similar look to Java, but be easier for non-programmers to work with.

  • object-oriented scripting language.
  • used to work with the objects associated with a web page document — the window, the document, the elements such as forms, images, links, etc.
  • originally developed by Netscape and called LiveScript
  • netscape collaborated with Sun Microsystems on modifications to the language and it was renamed JavaScript
  • JavaScript is NOT Java
  • common uses:
    • pop up, nested navigations
    • photo galleries
    • pre-validating form information
    • adding animated tool tips
    • place/read cookies (remember/retrieve previous states)
    • determine the status of user's browser (vendor, version, plugins, size, position)
    • display a message box
    • create a new window with a specified size and screen position
    • status messages
    • display current date
    • perform calculations

document object model (DOM)

The Document Object Model (DOM) is the mechanism through which javascript can manipulate an HTML page and access information about the user's browsing environment.

  • the document object model (DOM) is the collection of objects, properties and methods on a page
  • the DOM defines every object and element on a web page
  • javascript can use the DOM to determine the status of the browser, the HTML page content, and the CSS
  • through the DOM, javascript can also make changes, additions, or deletions to the HTML page content and CSS
  • its hierarchical structure can be used to access page elements and apply styles to page elements

Implementation:

  • place Javascript code as part of an event attached to an XHTML element (inline)
  • place Javascript code between <script> tags (embedded)
  • place Javascript code in an external file (external)
  • embedded script:

    <script type="text/javascript">
    alert("Welcome to Our Site");
    </script>

  • external script:

    <script type="text/javascript" src="myjavascript.js" ></script>

    A <script> tag does not allow using both Embedded and External javascript at the same time.

  • In HTML5, only the <script> tag is required, but to maximize backward compatability, the type="text/javascript" attribute may be included
  • Oftentimes, the best way to deploy your javascript is in external file(s). Using embedded javascript reduces reusability and maintainability.

jQuery

jQuery is written in javascript. It may contain instructions that use a jQuery specific syntax, and may also include standard javascript instructions.

  • jQuery is a JavaScript Library
  • Gives access to any DOM element
  • Designed especially for making JS manipulations easier
  • Fairly small footprint (smaller than an average image)
  • If you can access the element from CSS (level 1-3), you can access it from jQuery
  • Very little requirement to change any HTML
  • Syntax is a little scary at first, but easy to pick up
  • Very powerful
  • Very popular
  • Works in almost all browsers (including IE6! - Mostly)

      Three main forms:

      • Download from jQuery
  • In addition to the basic jQuery framework, you need to include your own code either as an embedded block or linked to an external file
  • Basic functions are easy to write

jQuery Basics

Although jQuery is itself written in javascript, it provides for several new features and techniques not available in 'vanilla' javascript.

Assigning Events

In javascript you would assign events to trigger functions by using the event attribute, eg:
<div onclick="alert('Hello World');" ></div>
In jQuery, events are not assigned in the HTML, but in the javascript code using a more object-oriented syntax (described below).

Variables

jQuery supports the use of an alternative syntax for variables. Instead of requiring the var keyword at definition, you can use a PHP style $ syntax. 

//standard javascript variables
var name01  = "Shania";
var age01	= 34;
document.write("Hello " + name01 ", you are " + age01 + " years old.");

//jQuery style variables
$name02   = "Jane";
$age02	= 29;
document.write("Hello " + $name02 ", you are " + $age02 + " years old.");

Patterns

You will notice this same pattern in many jQuery statements, 

 $(object).event(function(){
   //javascript code
 });
  • object: any valid selector
  • event: See a list of jQuery Events
  • function: this creates a new 'nameless', anonymous function that will be invoked as soon as the event occurs
  • javscript code: includes one or more javascript instructions, or jQuery instructions

The common syntactical patterns used by jQuery should be carefully noted. Great care must be exercised when coding to ensure the proper placement of a multitude of nested instructions.

jQuery Patterns

  • $(selector).event( );
    Most jQuery code will be contained inside this pattern. The selector is usually a valid CSS selector in quotes. The event is any one of the jQuery events

  • function(){}
    Anonymous functions are function definitions that have no name. They are automatically invoked and dop not require explicit invokation.

  • $(selector).event(  function(){}  );
    If you want to take action when an event occurs, you must insert an anonymous function into the parameter list section of the jQuery event.

Event Passing

  • If you need to know more about an event that has just occurred, define an event parameter for the anonymous function
  • Inside the function you can access the event as an object that you can query details upon
  • /* include a parameter in the anonymous function... */
$("div").click(function( event ){
	/* ...access event specific info using the parameter */
	alert("X position is " + event.pageX );
});

Overriding Default Behaviors

Sometimes you will want javascript behaviors to override the standard. For example, if you want to use the click event of an anchor tag, you may want to disable the usual anchor tag click.

Add the prevent.preventDefault();

$("a").click(function(event){
	event.preventDefault();
	alert("Hello World");
});

The basic pattern for applying effects is

Your instructor will walk you through some effect examples

Animate is especially powerful, and has a slightly more complex syntax:

$('selector').animate({
	property01: 'value',
	property02: 'value'
}, speed);

Note: animate can only effect CSS properties that have numeric values.

The Current Element (this)

  • You can use a dynamic selector called this to choose the element that most recently triggered an event
  • this refers to the current element
  • /* if a "div" is clicked... */
$("div").click(function(){
	/* ...you can use $(this) to choose the 
	specific div that was just clicked  */
	$(this).css('background-color', 'blue');
});
